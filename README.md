# TTUYDM - TeleTYpewriter + Text-based User Interface Display Manager

A Text User Interface (TUI) based Display Manager (DM).


## Dependencies:

* sh
* gcc
* make (gmake for BSD)
* cmake
* libpam
* doxygen (optional for documentation)
